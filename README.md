# Django task manager 

a simple backend rest API  for managing, tasks, built with Django and Django rest framework

## Features

- Create, view, update, delete tasks

## Endpoints

- `GET /api/tasks/`
- `POST /api/tasks/`
- `PATCH /api/tasks/<id>/`
- `DELETE /api/tasks/<id>/`
- `POST /api/summarise/`

## Tech Stack

- Python
- Django
- Django REST Framework
